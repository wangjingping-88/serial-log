从单个串口到多台电脑，常用操作都有明确的入口和状态反馈。

01 / OBSERVE

多个串口，同屏对照。

把中心、R1、R2 等窗口放在同一页面，连接状态和输出节奏一眼可见。窗口可以拖动、放大、排序，也可以隐藏命令区专注看日志。

01 / CONNECT

连接设备

刷新串口、设置波特率，连接单个或全部本地窗口。

02 / OPERATE

发送命令

在单条命令、命令导入或命令组之间切换，支持循环执行。

03 / TRACE

保存复盘

串口界面保持紧凑时间戳，完整年月日写入日志文件并自动分段。
